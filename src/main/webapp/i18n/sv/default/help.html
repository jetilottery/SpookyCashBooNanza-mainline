Spooky Boo-nanza

Gå in på kyrkogården om du vågar! Välkommen till Spooky Boo-nanza där kusliga gravstenar fulla av rikedomar väntar! Välj gravstenar för att avslöja priser eller bonussymboler – avslöja ett pris för att vinna direkt! Lås upp bonusar genom att samla in symboler för mer spöklika vinster – spela på orgeln i bonusen Orgeldonator för att samla in priser och multiplikatorer, eller öppna läskiga kistor i bonusen Kistandar för nervkittlande belöningar!

Så här satsar du

Lottkostnad

Visar den aktuella lottkostnaden.

Tryck på vänsterpilen (-) för att minska lottkostnaden.

Tryck på högerpilen (+) för att öka lottkostnaden.

Tryck på KÖP för att dra lottkostnaden från saldot och börja spela.

Så här spelar du

Välj 8 av 16 gravstenar för att avslöja prisbelopp eller bonussymboler.

Avslöja ett pris för att vinna det beloppet direkt!

Bonusen Orgeldonator

Avslöja tre orgelsymboler för att aktivera bonusen Orgeldonator. Välj en tangent på orgeln för att avslöja antingen ett prisbelopp, en multiplikator eller en Inkassera-symbol. Prisbeloppen summeras i slutet av bonusen och multipliceras med det slutliga multiplikatorvärdet. Avslöja en Inkassera-symbol för att avsluta bonusen.

Bonusen Kistandar

Avslöja tre kistsymboler för att aktivera bonusen Kistandar. Bonusen består av fem rundor med kistor, var och en med ökande prisvärden. Välj en av två kistor för att släppa ut andarna. Om antalet andar matchar eller överstiger markeringen på andemätaren för den rundan markeras priset som uppnåtts innan nästa runda tar vid. Varje runda kräver ett ökande antal andar för att avancera vidare och är beroende av vilket pris som erbjuds. Fortsätt avancera till nästa omgång tills du inte inkasserat det antal andar som krävs för aktuell omgång, eller tills du belönas med priset för den femte omgången.

Endast det högsta priset från den avslutade rundan tilldelas.

Priset och gränsvärdet för antalet andar som är knuten till varje runda kan variera mellan bonusspel.

Andar samlas ihop kumulativt under bonusen.

Avslöja automatiskt

Tryck på AVSLÖJA AUTOMATISKT för att automatiskt slutföra spelet. Spelet fortsätter tills alla val har gjorts.

Tryck på STOPP för att fortsätta spelet i manuellt spelläge.

Regler

Missbruk eller funktionsfel ogiltigförklarar alla utbetalningar och spel.

Bonusvinster läggs till det totala prisvärdet.

Direktvinster läggs till det totala prisvärdet.

Val som gjorts av spelare under spelets gång påverkar inte spelets utfall.

Vinster visas i vald valuta.
